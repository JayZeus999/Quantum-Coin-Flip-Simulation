# Quantum-Coin-Flip-Simulation

Built a Quantum Coin Flip simulator with Qiskit for #30DaysOfQuantum Project 2 
Initialized a qubit in |0⟩.
Applied Hadamard gate to invoke superposition (Heads + Tails at once).
Measured 1000+ shots to plot a ~50/50 distribution.
Then wrapped it into a function that samples from the quantum probability distribution and returns Heads or Tails on each call.

I had a bug where Matplotlib refused to work as usual.. placed %matplotlib inline at the top & voilà 🙂🙃
